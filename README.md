# StayTrack Real Estate Portal

StayTrack is a modern real estate web application built with React, Vite, and Tailwind CSS. It allows users to browse, search, and view detailed information about properties for sale. The platform features authentication, protected routes, advanced filtering, and a responsive, user-friendly interface.

## Features

- **Property Search & Filters:**  
  Users can filter properties by bedrooms, neighbourhood, price range, and sort results by price, area, or date.

- **Property Details:**  
  Each property has a dedicated detail page with images, price, features, agent contact info, and an embedded map.

- **Image Gallery:**  
  View property images in a modal gallery with carousel navigation and Picture-in-Picture support.

- **Authentication:**  
  Secure login system with protected routes for accessing property details and search pages.

- **Responsive Design:**  
  Fully responsive layout for desktop and mobile devices.

- **React Query Integration:**  
  Efficient data fetching and caching for property data.

- **Custom Toast Notifications:**  
  User feedback for actions like login, errors, and form validation.

- **Modern UI:**  
  Built with Tailwind CSS for a clean and customizable design.

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Query](https://tanstack.com/query/latest)
- [Axios](https://axios-http.com/)
- [React Router](https://reactrouter.com/)
- [Strapi API](https://strapi.io/) (for property data)

## Getting Started

1. **Clone the Repository**

  ```bash
  git clone https://github.com/jeevabarathishivakumar/stayview.git
  ```

2.  **Navigate to the Project Folder**
   ```bash
    cd staytrack-real-estate-machine-test
    ```

3. **Install dependencies:**
   ```bash
   npm install
   ```


4. **Start the Development Server**

Run the Vite development server:

```bash
npm run dev
```

The application will start and be available at:

```
http://localhost:5173
```
